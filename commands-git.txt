 commands-git: 
Cloner un repository : 
-git clone lien d'accès ou adresse du repository

Les trois commandes push : 
-git push
-git push origin <branche> soit main soit master
-git push -u origin <branche> soit main soit master

Commande qui permet de pull : 
-git pull origin main ou master

Commande pour créer une nouvelle branche :
-git branch "nom de la branche"

Commande pour switcher de branche : 
git checkout "nom de la branche"
